@ianwalter/sled

A JavaScript micro-library for animating an element's height to create a slide effect

About

I created Sled because I wanted to use Bootstrap's collapsible navbar menu without having to import a bunch of unnecessary code. Sled is 20 lines of code, 1235 bytes (595 gzipped), and has no dependencies.

Demo

Click the button below to toggle between sliding the menu down and up.

Installation

          
npm install @ianwalter/sled --save

Usage

Vanilla JavaScript example: 
          
import Sled from '@ianwalter/sled'
const sled = new Sled(document.querySelector('#myEl'))

// Slide to a specific height:
sled.slide('100px')

// Toggle between 0 height and the elements natural height.
sled.toggle()
Vue.js example: 
          
import Sled from '@ianwalter/sled'

let sled

export default {
  mounted () {
    sled = new Sled(this.$refs.menu)
  },
  methods: {
    toggleMenu: () => sled.toggle()
  }
}
